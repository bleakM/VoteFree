# VoteFree 问卷系统（局域网 + 单机离线）

VoteFree 是一个本地运行的中型 Python 应用，包含：

- 管理员桌面端（CustomTkinter）
- 局域网 HTML 问卷服务（Flask）
- 离线 HTML 票据生成模式（单机 `.vote`）
- `.vote` 加密存储与历史归票统计

## 主要能力

- 问卷管理（展板模式）：题目按真实作答顺序从上到下排列，支持拖动排序与上移下移
- 题型支持：`single/multi/rating/slider/text/textarea`
- 循环块：循环只在“循环块”中启用，支持按名单、列表对象、必填多选结果循环
- 高级逻辑：右侧逻辑面板支持显示条件、必填条件、循环筛选（全部/自评/互评）
- 联合限制：支持问卷级联合规则（可视化配置，无需写 JSON；多题求和、阈值计数、选项命中计数、跨题比较）
- 安全访问：问卷口令校验（口令正确才可作答）
- 身份策略：仅实名流程，进入问卷前采集身份字段（字段可自定义、全部必填）
- 名单联动：绑定名单后自动生成名单列表对象，可用于循环与条件
- 新手引导：内置 5 步流程面板、快速跳转、一键创建示例名单与示例问卷
- 模板与诊断：问卷管理页内可直接“套用模板”并继续逐题修改，提供“配置体检”
- 局域网模式：管理员本机启动服务，同 LAN 设备访问 `http://IP:端口/q/问卷ID`
- 局域网扫码：服务页自动生成二维码，手机/平板扫码直达问卷
- 单机离线模式：导出单文件 HTML，填写后生成 `.vote` 文件，交管理员导入归票
- 加密票据：统一 `.vote` 格式，使用 `RSA-OAEP-3072 + AES-256-GCM`
- 历史追溯：管理员解密查看历史票据并进行 SQL 分析
- SQL 查询统计：票据与SQL页提供 SQL 工作台，支持多条 `SELECT` 语句、命令行风格显示、按结果集导出 CSV
- 查询模板：可按问卷保存/加载 SQL 模板，且与问卷模板绑定（同模板新建问卷可自动带出 SQL 模板）

## 安全模型

- 首次启动要求设置管理员密码（本地保存 PBKDF2 哈希）
- 同时生成 RSA 密钥对：
  - 公钥用于加密新票据
  - 私钥使用管理员密码加密存储
- `.vote` 票据为混合加密信封，未解锁私钥不可读取内容

## 快速开始

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python main.py
```

可选：启动浏览器内核管理端（迁移中）

```powershell
python main.py --web-shell
```

首次启动：
1. 设置管理员密码
2. 先打开“新手引导”页，按 5 步流程操作
3. 在“问卷管理”使用“展板模式”设计问卷（可先套用模板）
4. 在“局域网服务”页启动服务或在“离线模式”导出 HTML

## 展板式问卷设计（新）

1. 左侧选择历史问卷，或点击“新建空白”。
2. 中间展板点击“添加普通题/添加循环块”。
3. 在题目卡片里直接改题干、题型、选项和限制项。
4. 循环题请放入“循环块”，并设置循环依据（名单/列表/多选题结果）。
5. 右侧“逻辑面板”可写入显示条件、必填条件与循环筛选。
6. 点击“联合规则”可通过表单配置多题联合限制规则（无需写代码）。
7. 点击“配置体检”检查错误，再点击“保存问卷”。
8. 可使用“专注设计/显示两侧”一键切换大画布模式。

题目灵活编辑能力：
- 支持普通题与循环块内题目的独立配置
- 多选题支持最少/最多选择，滑杆题支持范围与步进
- 文本题支持长度/词数/行数限制
- 支持拖动排序 + 上下移动 + 复制 + 删除
- 支持自评/互评筛选（循环块内题目）
- 支持问卷级联合规则（手动配置 + 示例模板）


- `dist\VoteFree\VoteFree.exe`

## `.vote` 文件格式（摘要）

`.vote` 是 JSON 信封，关键字段：

- `algorithm`: `RSA-OAEP-3072+AES-256-GCM`
- `encrypted_key_b64`: 被 RSA 加密后的会话密钥
- `nonce_b64`: AES-GCM 随机 IV
- `ciphertext_b64`: 加密后的票据正文

正文（解密后）包含：

- `submission_id`
- `questionnaire_id`
- `respondent`
- `context`（关系类型/目标对象）
- `answers`
