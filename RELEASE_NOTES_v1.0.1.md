# VoteFree v1.0.1

发布日期：2026-03-17

## 修复内容
- 修复了 Web 内核导出离线问卷时未注入名单校验成员数据的问题。
- 修复了离线问卷在名单校验模式下，即使输入正确信息仍提示“不在允许名单中”的问题。
- 统一 Tk/Web 两套离线导出链路，降低前后端行为不一致风险。

## 下载说明
- `VoteFree-v1.0.1-windows-x64.zip`：Windows 可执行版本（默认运行包）。
- `VoteFree-v1.0.1-blank.zip`：空白发布包（不含业务数据，含可独立运行源码 `src/`）。

## 升级建议
1. 升级前备份旧版 `data/`。
2. 替换新程序后恢复 `data/`。
3. 重新导出离线 HTML 后再分发给填报者。
